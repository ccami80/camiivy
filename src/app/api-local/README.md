# api-local

프론트 자체해결 API용 Next.js API Routes가 들어가는 폴더입니다.

- **용도**: DB·타사 연동 없이 프론트에서 해결하는 로직. 서버에서 간단한 연산·포맷·검증만 하거나, 클라이언트 요청을 그대로 처리하는 경로.
- **경로**: `/api-local/...` 형태로 노출
- **예**: 클라이언트 편의용 엔드포인트, 쿠키/헤더 기반 간단 응답, 정적/계산 결과 반환 등

자사 백엔드 API는 `src/app/api/`, 타사 API는 `src/app/api-etc/`, 목업은 `src/app/api-mock/`를 사용하세요.
